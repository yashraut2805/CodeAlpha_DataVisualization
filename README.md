# Student Performance Data Visualization

## Project Overview
This project focuses on Data Visualization using the Students Performance Dataset.

The project analyzes student scores and visualizes important insights using charts and graphs.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Average score analysis
- Gender distribution visualization
- Parental education impact analysis
- Correlation heatmap

This project helped improve my understanding of:
- Data visualization
- Graph plotting
- Python libraries
- Data analysis techniques
